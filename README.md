# PRODIGY_GA_02 – Text-to-Image Generation using Stable Diffusion

## Introduction

As part of my Generative AI Internship at Prodigy InfoTech, I developed a Text-to-Image Generation System using the Stable Diffusion model. This project demonstrates how Artificial Intelligence can transform simple text descriptions into visually appealing and meaningful images.
The project provided hands-on experience in Generative AI, Diffusion Models, and Prompt Engineering while exploring real-world applications of AI-powered content creation.

## Project Objective
The objective of this project is to generate high-quality images from natural language prompts using a pre-trained Stable Diffusion model.

The model interprets textual descriptions and creates unique images that represent the given prompt, showcasing the creative capabilities of Generative AI.

## Workflow

1. Install the required libraries and dependencies.
2. Load the Stable Diffusion pre-trained model.
3. Provide a text prompt as input.
4. Generate an image using the model.
5. Save and analyze the generated output.
